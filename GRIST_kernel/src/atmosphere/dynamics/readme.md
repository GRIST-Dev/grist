## Key Compiler Options

- `DYCORE_NHD`:  
  - A new option to separate **NDC** at compile time.  
  - Use `-DDYCORE_NHD` to compile **NDC**.  
  - Without this option, only **HDC** is compiled.  

  **Definitions:**  
  - **HDC** = Hydrostatic Dynamical Core  
  - **NDC** = Nonhydrostatic Dynamical Core
